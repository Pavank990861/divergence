## Getting Started

### Installation
No installation required! 

**Quick Start Options:**
1. **GitHub Pages**: Visit the [Live Demo](https://pavank990861.github.io/divergence/advanced-editor/) link above
2. **Local**: Download/clone the repository and open `index.html` in your browser
3. **Direct File**: Right-click `index.html` â†’ Open with â†’ Your preferred browser

### Usage

1. **View Dashboard**: Open `index.html` to see all your content
2. **Create Content**: Click "Create New" button
3. **Edit Content**: Click on any content card or use the edit button
4. **Search**: Use the search bar to find content
5. **Filter**: Use category and sort dropdowns to organize content
6. **Save**: Click "Save" button to save your content
7. **Preview**: Click "Preview" to see how your content looks

## Features in Detail

### Rich Text Editor
- Format text with toolbar buttons
- Insert links and images
- Create lists (ordered and unordered)
- Add headings (H1, H2)
- Code blocks and quotes
- Undo/Redo functionality

### Content Management
- All content stored locally in browser
- No server required
- Data persists across sessions
- Export/import capabilities (can be added)

### Search & Filter
- Real-time search across titles and content
- Filter by category (Document, Note, Article, Draft)
- Sort by: Newest, Oldest, Title, Recently Modified
